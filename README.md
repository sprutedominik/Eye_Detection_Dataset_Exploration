# Eye Detection Dataset Analysis

Exploratory computer vision analysis of a human eye detection dataset using Python and OpenCV.

## Project Overview

This notebook explores a labeled eye detection dataset with annotations in YOLO format.

The analysis includes:

- Inspecting the dataset structure
- Loading images with OpenCV
- Parsing YOLO annotation files
- Visualizing bounding boxes on eye images
- Verifying annotation quality

## Tools

Python  
OpenCV  
NumPy  
Matplotlib

## Dataset

Eye Detection Dataset (Kaggle)

Contains:
- 1385 training images
- YOLO formatted bounding box labels

## Example Output

Bounding boxes are visualized on the images to confirm that annotations align correctly with the eye region.
